# Claude Code Skill 练习项目

此项目用于练习和测试 Claude Code 的 skill 功能开发。

## 项目目的

本仓库专门用于练习编写 Claude Code skills，包含多个不同类型的技能示例和开发工具。

## 目录结构

- `weekly-report-workspace/` - 毕业论文周报技能的工作区
- `find-skills/` - 技能发现功能相关文件
- `frontend-design-2/` - 前端设计技能相关文件
- `skill-creator/` - 技能创建和管理工具集
- `evals/` - 技能评估文件
- `scripts/` - 辅助脚本（如 md 到 PDF 转换）

## 文件说明

- `毕业论文周报（3月24日-3月30日）.md` - 示例周报文件，用于测试 weekly-report 技能
- `CLAUDE.md` - Claude Code 配置文件
- `SKILL.md` - 技能定义文件
- 各目录中的许可证、代理、脚本和其他配置文件

## 使用方法

此项目展示了如何创建、修改和测试 Claude Code skills，特别适用于需要自动化生成标准格式报告的场景。

## 注意事项

- 主要用于技能开发和测试目的
- 包含多种技能类型示例用于学习和参考
- 所有技能均遵循 Claude Code 标准规范
